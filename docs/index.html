What It Does

Give it a script and your footage. It handles the rest. The server parses your script, scans your media library, generates an edit decision list, and assembles the timeline in Premiere Pro -- all from a single prompt.

Timeline Editing

Insert, overwrite, trim, split, slip, slide, ripple, and roll clips. Full timeline assembly from EDL, CSV, or AI-generated edit lists.

Color Grading

Full Lumetri Color control: exposure, contrast, color wheels, curves, HSL secondaries, LUTs, vignette, and auto color matching across clips.

Audio Mixing

62 audio tools for levels, EQ, compression, limiting, de-essing, loudness normalization, Essential Sound panel, and waveform analysis.

Effects & Transitions

Apply, configure, and keyframe any effect or transition. Built-in presets for film looks, motion graphics, and visual effects.

AI-Powered Workflows

Smart rough cuts, auto B-roll suggestions, script-to-timeline, social media exports, and intelligent clip analysis powered by AI.

Export & Delivery

Direct export, Adobe Media Encoder queue, codec conversion (ProRes, H.264/265, DNxHR), batch exports, and delivery checks.

Architecture

Four languages, each playing to their strengths:

Go Orchestrator

MCP protocol handling, concurrency, service mesh, health checks, and circuit breakers. The central hub that routes all tool calls.

Rust Media Engine

High-performance media scanning, waveform analysis, thumbnail generation, and real-time processing. When speed matters.

Python Intelligence

Script parsing, EDL generation, NLP, AI-powered editing decisions, and smart clip analysis. The brain behind the automation.

TypeScript Bridge

Connects to Premiere Pro via CEP/ExtendScript. Translates MCP tool calls into native Premiere Pro operations.

Quick Start

git clone https://github.com/ayushozha/AdobePremiereProMCP.git
cd AdobePremiereProMCP
just install && just build
just install-panel  # Install CEP panel into Premiere Pro
just start          # Start all services

Or double-click PremierPro.command (macOS) / PremierPro.bat (Windows) for one-click launch.

Supported Versions

Works with Adobe Premiere Pro 2020 through 2026 on macOS and Windows. The bridge uses Adobe's CEP (Common Extensibility Platform) and ExtendScript.

Premiere Pro 2020-2022

Community tested. Core features supported.

Premiere Pro 2023-2024

Fully supported. All tools verified.

Premiere Pro 2025-2026

Primary target. Full compatibility.
